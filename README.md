# nippon-color

Traditional Japanese color palettes for JavaScript and TypeScript. ~250 colors from [nipponcolors.com](https://nipponcolors.com/).

| Package | Description |
|---|---|
| `@nippon-color/core` | Typed JS/TS constants |
| `@nippon-color/css` | CSS utility classes |
| `@nippon-color/tailwind` | Tailwind CSS plugin |

See [docs](./docs) for full usage.
